# Changelog

All notable changes to PLANY will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-28

### Added
- Initial release of PLANY
- User authentication with Firebase Auth (email/password)
- Lesson planning with full CRUD operations
- Monthly calendar view with lesson indicators
- Weekly schedule builder with drag-and-drop functionality
- Lesson templates system
- School year configuration
- Holiday and break period management
- Real-time data synchronization with Firestore
- Responsive design for mobile and desktop
- User profile management
- Lesson copying/duplication feature
- Firebase Hosting deployment configuration
- Comprehensive documentation (README, DEPLOYMENT, CONTRIBUTING)

### Features
- **Calendar**: Visual monthly calendar showing planned lessons
- **Weekly Schedule**: Interactive 6-period x 5-day timetable
- **Lessons**: Create, edit, delete, and duplicate lessons
- **Templates**: Pre-built and custom lesson templates
- **School Year**: Configure academic year settings and breaks
- **Cloud Sync**: Automatic synchronization across devices
- **Authentication**: Secure user accounts with Firebase

### Technical
- Built with Vanilla JavaScript ES6+
- Firebase/Firestore for backend
- Vite for build tooling
- CSS Custom Properties for theming
- Mobile-first responsive design

### Security
- Firestore security rules ensuring user data privacy
- Email/password authentication
- User-specific data isolation
